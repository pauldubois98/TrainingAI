Neural Networks by Hand

Build and explore neural networks neuron by neuron.

Tool

How to use the tool

  • Click an input neuron to cycle through its allowed values.
  • Drag the sliders at the bottom to set each weight and bias.
    • Connection lines grow thicker and greener (positive) or redder (negative) as weights change.
    • A coloured outline on a neuron shows its bias magnitude.
  • Hoover any non-input neuron to see the full calculation that produced its output.
  • Use first dropdown to change the activation function.
  • Use second dropdown to change the allowed values.
  • Use inputs to adjust the number of neurons on each layer.

Challenges
